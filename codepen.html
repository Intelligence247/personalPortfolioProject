These are exercises to help me fruther my front-end coding skills and were created for fun. If you are looking to get started with creating your own illustrations, I created a post here which may help you.

Responsive Devices

I originally created this for my old website because I needed something to grab the attention of users when they first came to the website. I decided to create Apple devices using only two HTML elements. When the user resized the browser, the device would change. No SVG was used and all illustrations and animations were done in-browser while it was being coded.
